24 * @brief 2D grid with contiguous memory layout matching Bellhop's internal

26 *
27 * @par Memory layout:
28 * - Coordinates: SoA (separate xCoords, yCoords arrays)
29 * - Values: Single flat array, row-major order
30 * - data[ix * ny + iy] = value at (ix, iy)
31 *
32 * @par Index order:
33 * - index(ix, iy) = ix * ny + iy (row-major)
34 * - Must match Bellhop's Fortran layout when interfacing
35 *
36 * @par Invariants:
37 * - xCoords.size() > 0 && yCoords.size() > 0
38 * - data.size() == nx() * ny()
