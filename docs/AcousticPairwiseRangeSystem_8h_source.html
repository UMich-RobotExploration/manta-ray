88/**
89 * @brief Manages pairwise acoustic ranging between all robots and landmarks.
90 *
91 * @details Builds directed links for every robot-to-robot and robot-to-landmark
92 * pair, runs Bellhop ray tracing to obtain time-of-flight, and converts TOF to
93 * range using the local sound speed profile. Robot-robot pairs exploit acoustic
94 * reciprocity to halve the number of Bellhop runs.
95 *
96 * Measurements are accumulated in a flat chronological log accessible via
97 * getMeasurements(). A lightweight checkBounds() method can be called at
98 * physics-rate to cull out-of-bounds robots without running Bellhop.
99 *
100 * @section iterative_beam_solver Iterative Beam Refinement
101 *
102 * When acquiring time-of-flight, the system uses direct-path-only filtering
103 * (zero surface and bottom bounces). If no direct-path arrival is found at the
104 * initial beam count, an iterative solver increases the beam density and
105 * re-runs Bellhop until either a direct path is found or the maximum beam
106 * count is reached.
107 *
108 * The beam count is scaled by kBeamIterativeFactor on each iteration and
109 * clamped to `AcousticsBuilder::getMaxBeams()` so the final iteration always
110 * runs at full allocated resolution, even when the max is not a clean multiple
111 * of the scale factor. After the loop completes (success or exhaustion), the
112 * beam count is restored to its original value for subsequent links.
113 *
114 * Example progression with `numBeams=80`, `maxBeams=300`,
115 * `kBeamIterativeFactor=2.0`:
116 * @code
117 * Step 1: 80 beams → no direct path
118 * Step 2: 160 beams → no direct path
119 * Step 3: 300 beams → clamped from 320, final attempt
120 * @endcode
121 *
122 * Configuration (via JSON `"acoustics"` block):
123 * - `num_beams`: initial beam count per axis (default 80)
124 * - `max_beams`: maximum beam count for iterative refinement (default 180)
125 * - `beam_spread_deg`: half-cone angle in degrees (default 20.0)
126 *
127 * @see AcousticsBuilder::rebuildBeam(), AcousticsBuilder::getMaxBeams()
128 * @see @ref iterative_beam_refinement "Iterative Beam Refinement (sim.md)"
129 */
